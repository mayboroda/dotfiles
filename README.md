# .dotfiles

My personal setup for .dotfiles

## Install
- install homebrew
- install GPG
- install git
- clone the repo
- run ./sync.sh

## Colors
- Main color palletes is [NordTheme](https://www.nordtheme.com/)
- iTerm2 Nord pallete by [arcticicestudio](https://github.com/arcticicestudio/nord-iterm2)
